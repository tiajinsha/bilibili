## ✨ 特性

- ✅ 使用 react + Typescript 全家桶开发
- 📺 支持bilibili视频播放和直播
- 📃 支持鼠标悬停预览视频效果
- 📃 支持B站特色弹幕
- 🛠 electron 后台线程集成 bilibili-api。
- 🖥️ 方便二次开发，其他功能开发中

# 安装依赖

yarn install

# 运行

yarn dev

# 打包

yarn build
